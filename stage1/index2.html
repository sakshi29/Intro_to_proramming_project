DOM - Docment Object Model

It define the logical structure of documents and the way a document can be accessed and manipulated. It is a tree like structure. where root is at top and leaf nodes are at bottom.

Boxes

HTML page is a collection of boxes or we can divide layout into many different rectangular boxes. It makes coding easy. And make document structured.

CSS

CSS stands for cascading style sheet. CSS is used to add style to the HTML document. CSS control the layout design of HTML documents all at once.
A CSS comprises of style rules that are interpreted by the browser and then applied to the corresponding elements in your document. A style rule is made of three parts −

Selector − A selector is an HTML tag at which a style will be applied. This could be any tag like <h1> or <table> etc. There are different type of selector

Property - A property is a type of attribute of HTML tag. Put simply, all the HTML attributes are converted into CSS properties. like color, border etc.

Value - Values are assigned to properties.

selector { property: value }

How to include CSS in HTML

There are 3 methods for adding CSS in HTML document.

Internal Style Sheet: can be defined within <style> tag inside the <head> section of HTML page.

<head>
   <style>

    body {

            background-color: linen;

             }

   </style>
</head>

External Style Sheet: A reference to external CSS style sheet can be added to the HTML page. The advantage of using external style sheet is that more than one page can be styled using same CSS. No need to write again and again same code.

<head>

   <link rel="stylesheet" type="text/css" href="mystyle.css">

</head>

Inline Style Sheet: An inline style may be used to apply a unique style for a single element.

< h1 style="color:blue; margin-left:30px;" >This is a heading.</h1>
